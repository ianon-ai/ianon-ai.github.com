<h2 id="faq-general">General</h2>

<div class="faq-list">

<details class="faq-item">
  <summary>What is ianon?</summary>
  <div class="faq-answer">
    <p>ianon is a local desktop AI chat application that lets you manage and interact with multiple AI providers from a single interface, while routing traffic through the Anyone Protocol anonymity network.</p>
    <p>Instead of juggling separate tabs, accounts, and interfaces for multiple providers, ianon puts everything in one place. You can add your API keys, organize providers and models, and switch between them easily.</p>
    <p>Broadcast mode lets you send the same prompt to every provider at once and compare their responses side by side.</p>
  </div>
</details>

<details class="faq-item">
  <summary>Which AI providers are supported?</summary>
  <div class="faq-answer">
    <p>On install, ianon supports more than 24 AI providers and allows more providers to be added if needed.</p>
    <p>You can chat with a single AI provider or broadcast to all enabled providers at the same time.</p>
  </div>
</details>

<details class="faq-item">
  <summary>Is ianon free?</summary>
  <div class="faq-answer">
    <p>Yes. The app is free to install.</p>
    <p>ianon does not include API keys for AI providers. To use provider APIs, you need to add your own API keys.</p>
  </div>
</details>

</div>

<h2 id="faq-setup">Setup & Usage</h2>

<div class="faq-list">

<details class="faq-item">
  <summary>Do I need my own API keys?</summary>
  <div class="faq-answer">
    <p>Yes. To use ianon for free, you need to add your own API keys for whichever providers you want to use.</p>
    <p>A paid model that provides access to a wide range of providers and models is planned.</p>
  </div>
</details>

<details class="faq-item">
  <summary>What platforms are supported?</summary>
  <div class="faq-answer">
    <p>ianon currently supports macOS Apple Silicon and macOS Intel.</p>
    <p>Windows and Linux support is planned.</p>
  </div>
</details>

<details class="faq-item">
  <summary>What is Broadcast mode?</summary>
  <div class="faq-answer">
    <p>Broadcast mode sends your message to all enabled AI providers at once and shows their responses side by side.</p>
    <p>This is useful for comparing outputs or getting multiple perspectives on the same prompt.</p>
  </div>
</details>

<details class="faq-item">
  <summary>Can I use ianon without the Anyone Network?</summary>
  <div class="faq-answer">
    <p>No. Routing through the Anyone Protocol is core to the app.</p>
    <p>There is no direct-connection mode, and the app does not send data outside of the Anyone Network.</p>
  </div>
</details>

</div>

<h2 id="faq-privacy">Privacy</h2>

<div class="faq-list">

<details class="faq-item">
  <summary>Do I need an account to use the app?</summary>
  <div class="faq-answer">
    <p>No. There is no registration, email, or login.</p>
    <p>You only need valid API keys for the providers you want to use.</p>
  </div>
</details>

<details class="faq-item">
  <summary>Does ianon send any data to a third-party provider?</summary>
  <div class="faq-answer">
    <p>Only your API request, prompt, and model parameters are sent to the LLM provider you select.</p>
    <p>These requests are routed through the Anyone Protocol network.</p>
    <p>No analytics, telemetry, or extra metadata is sent anywhere else.</p>
  </div>
</details>

<details class="faq-item">
  <summary>Where is my data stored?</summary>
  <div class="faq-answer">
    <p>Your data is stored locally on your machine.</p>
    <p>Chat history, API keys, and settings are written to your operating system’s user data directory.</p>
    <p>Nothing is synced, uploaded, or backed up externally.</p>
  </div>
</details>

<details class="faq-item">
  <summary>Can I delete my chat history?</summary>
  <div class="faq-answer">
    <p>Yes. All data is stored locally, so you have full control.</p>
    <p>You can clear individual chat history or delete all chat history from within the app.</p>
  </div>
</details>

</div>

<h2 id="faq-prerequisites">Prerequisites</h2>

<div class="faq-list">

<details class="faq-item">
  <summary>What are the requirements to install and use ianon?</summary>
  <div class="faq-answer">
    <p>You need:</p>
    <ul>
      <li>4GB RAM</li>
      <li>Around 500MB disk space</li>
      <li>An active internet connection</li>
      <li>At least one valid API key from a supported AI provider</li>
    </ul>
  </div>
</details>
