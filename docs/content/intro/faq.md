# FAQ

## What is ianon?

iAnon is a local desktop AI chat application that lets you manage and interact with multiple AI providers from a single interface, while routing all traffic through the [Anyone Protocol](https://www.anyone.io) anonymity network.

Instead of juggling separate tabs, accounts, and interfaces for multiple providers, iAnon puts everything in one place. You can add your API keys, organize providers and models, and switch between them easily.

Broadcast mode lets you send the same prompt to every provider at once and compare their responses side by side.



## Which AI providers are supported?

On install, iAnon supports more than 24 AI providers and allows more providers to be added if needed.

You can chat with a single AI provider or broadcast to all enabled providers at the same time.



## Is iAnon free?

Yes. The app is free to install.

iAnon does not include API keys for AI providers. To use provider APIs, you need to add your own API keys.

# Setup & Usage

## Do I need my own API keys?

Yes. To use iAnon for free, you need to add your own API keys for whichever providers you want to use.

A paid model that provides access to a wide range of providers and models is planned.



## What platforms are supported?

iAnon currently supports:

- macOS Apple Silicon
- macOS Intel

Windows and Linux support is planned.



## What is Broadcast mode?

Broadcast mode sends your message to all enabled AI providers at once and shows their responses side by side.

This is useful for comparing outputs or getting multiple perspectives on the same prompt.



## Can I use iAnon without the Anyone Network?

No. Routing through the Anyone Protocol is core to the app.

There is no direct-connection mode, and the app does not send data outside of the Anyone Network.

# Privacy

## Do I need an account to use the app?

No. There is no registration, email, or login.

You only need valid API keys for the providers you want to use.



## Does iAnon send any data to a third-party provider?

Only your API request, prompt, and model parameters are sent to the LLM provider you select.

These requests are routed through the Anyone Protocol network.

No analytics, telemetry, or extra metadata is sent anywhere else.



## Where is my data stored?

Your data is stored locally on your machine.

Chat history, API keys, and settings are written to your operating system’s user data directory.

Nothing is synced, uploaded, or backed up externally.



## Can I delete my chat history?

Yes. All data is stored locally, so you have full control. 
You can clear individual chat history or delete all chat history from within the app.

# Prerequisites

## What are the requirements to install and use iAnon?

You need:

- 4GB RAM
- Around 500MB disk space
- An active internet connection
- At least one valid API key from a supported AI provider
