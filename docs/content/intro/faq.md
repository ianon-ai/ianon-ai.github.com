## What is iAnon?

iAnon is a local desktop AI chat application that lets you manage and interact with multiple AI providers from a single interface while routing all traffic through the [Anyone Protocol](https://www.anyone.io) anonymity network.

Instead of juggling separate tabs, accounts, and interfaces for multiple providers, the iAnon app puts them all in one place. Add your API keys, organize providers and models and switch between them effortlessly.&#x20;

Broadcast mode lets you send the same prompt to every provider at once and compare their responses side by side.

### Which AI providers are supported?

Upon install iAnon supports more than 24+ AI providers and supports adding more providers if needed. You can chat with any single AI provider or broadcast to all of them simultaneously.

### Is iAnon free?

Yes. The app is free to install and does not include API keys for any AI providers.

## Setup & Usage

### Do I need my own API keys?

Yes. To use it for free you are required to add your own API keys for whichever providers you want to use. A paid model providing access to wide range of providers and models is coming.

### What platforms are supported?

macOS (Apple Silicon and Intel). Windows and Linux support is planned.

### What is Broadcast mode?

"Broadcast all" sends your message to all enabled AI providers at once and shows their responses side by side. Useful for comparing outputs or getting multiple perspectives.

### Can I use it without the Anyone Network?

No. Routing through Anyone Protocol is core to the app. There is no direct-connection mode and the app does never send any data outside of Anyone Network

## Privacy

### **Do I need an account to use the app?**&#x20;

No. No registration, no email, no login. You only need valid API keys for the providers you want to use.

### **Does iAnon send any data to a third-party provider?**&#x20;

Only your API requests, the prompt and model parameters are sent to the LLM provider you selected, routed through the Anyone Protocol network. No analytics, telemetry, or metadata is sent anywhere else.

### **Where is my data stored?**&#x20;

Locally on your machine. Chat history, API keys, and settings are written to your OS user data directory. Nothing is synced, uploaded, or backed up externally.

### **Can I delete my chat history?**&#x20;

Yes. All data is stored locally so you have full control. You can clear individual chat history or all at once from within the app.

## Prerequisites&#x20;

### **What are the requirements to install and use iAnon?**&#x20;

* 4GB RAM.
* \~500MB disk space.&#x20;
* An active internet connection is required.
* At least one valid API key from a supported AI provider.
